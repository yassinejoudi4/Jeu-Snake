# Jeu Snake

Petit projet de mise à niveau en programmation, réalisé en langage C.  

## Fonctionnalités

- Déplacement du serpent avec le clavier.  
- Gestion des collisions avec les murs et le corps du serpent.  
- Calcul du score en fonction de la longueur du serpent.  
- Interface graphique simple utilisant la bibliothèque SDL.  

## Technologies

- Langage : C  
- Bibliothèque graphique : SDL  
- Compilateur : GCC  

## Objectif

- Pratiquer les bases de la programmation en C et la manipulation d’une interface graphique simple.
